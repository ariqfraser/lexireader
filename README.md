# LexiReader

### How to install:

1. Clone this repository.
2. Load the extension on Chrome (or other Chromium browsers):
   1. Access `chrome://extensions/`
   2. Enable `Developer mode`
   3. Click on `Load unpacked extension`
   4. Select the `build` folder.
